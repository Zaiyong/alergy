# Assets Directory

This directory should contain the following Expo asset files:

- `icon.png` - App icon (1024x1024px)
- `splash.png` - Splash screen image (1242x2436px recommended)
- `adaptive-icon.png` - Android adaptive icon (1024x1024px)
- `favicon.png` - Web favicon (48x48px)

You can generate these using:
- [Expo Asset Generator](https://www.npmjs.com/package/expo-asset-generator)
- Or create them manually according to Expo's asset guidelines

For now, placeholder files are fine for development. Replace them before production.
